# Slash Commands для ChinaBot

## Обзор

Проект был обновлен для поддержки Discord Slash Commands вместо команд с префиксом "!". Это обеспечивает лучший пользовательский опыт с автодополнением и встроенной справкой.

## Что изменилось

### 1. Новые файлы
- `bot/slash_commands.go` - Регистрация и обработка slash commands
- `bot/interaction_helpers.go` - Вспомогательные функции для взаимодействий
- `SLASH_COMMANDS_README.md` - Данная документация

### 2. Обновленные файлы
- `bot/handlers.go` - Добавлена поддержка slash commands в обработчике взаимодействий
- `bot/discord.go` - Автоматическая регистрация slash commands при запуске
- `ranking/commands.go` - Добавлены новые обработчики для slash commands
- `ranking/ranking.go` - Добавлен метод `GetPolls()` для автодополнения
- `ranking/kki.go` - Добавлены методы `GetNFTs()` и `GetCases()` для автодополнения

## Доступные Slash Commands

### Основные команды
- `/china [user]` - Показать баланс соцкредитов
- `/top` - Показать топ-5 пользователей
- `/stats [user]` - Показать статистику пользователя
- `/transfer user amount [reason]` - Передать соцкредиты
- `/help` - Показать справку по командам

### Игровые команды
- `/rb [color] [bet]` - Игра Красный-Чёрный
- `/blackjack [bet]` - Игра Блэкджек
- `/duel user bet` - Вызвать на дуэль

### NFT и кейсы
- `/inventory` - Показать инвентарь NFT
- `/case_inventory` - Показать инвентарь кейсов
- `/case_bank` - Показать банк кейсов
- `/daily_case` - Получить ежедневный кейс
- `/open_case case_id` - Открыть кейс (с автодополнением)
- `/sell nft_id count` - Продать NFT (с автодополнением)
- `/trade_nft user nft_id count` - Передать NFT (с автодополнением)
- `/show_nft nft_id` - Показать информацию о NFT (с автодополнением)
- `/buy_case_bank case_id count` - Купить кейс из банка (с автодополнением)
- `/case_trade user case_id count` - Купить кейс у игрока (с автодополнением)

### Экономика
- `/btc` - Показать курс биткойна
- `/prices` - Показать статистику цен NFT

### Опросы
- `/polls` - Показать активные опросы
- `/cpoll question options` - Создать опрос (только админы)
- `/dep poll_id option amount` - Сделать ставку на опрос (с автодополнением)

### Киноаукцион
- `/cinema title amount` - Предложить вариант для киноаукциона
- `/cinema_list` - Показать варианты киноаукциона
- `/bet_cinema option amount` - Сделать ставку на киноаукцион

### Админские команды
- `/admin user amount [reason]` - Изменить баланс пользователя
- `/admin_mass operation amount users [reason]` - Массовое изменение баланса
- `/sync_nfts` - Синхронизировать NFT из Google Sheets
- `/admin_give_case user case_id` - Выдать кейс (с автодополнением)
- `/admin_give_nft user nft_id count` - Выдать NFT (с автодополнением)
- `/admin_remove_nft user nft_id count` - Удалить NFT (с автодополнением)
- `/admin_refresh_bank` - Обновить банк кейсов
- `/admin_reset_limits` - Сбросить лимиты кейсов
- `/admin_cinema_list` - Детальный список киноаукциона
- `/admin_remove_lowest count` - Удалить самые низкие варианты киноаукциона
- `/admin_adjust_cinema option operation amount` - Корректировать сумму варианта
- `/admin_remove_cinema user option` - Удалить вариант киноаукциона
- `/admin_end_blackjack user` - Завершить игру в блэкджек
- `/admin_holiday_case user count` - Выдать праздничный кейс
- `/admin_holiday_case_all count` - Выдать праздничный кейс всем
- `/admin_close_dep poll_id winning_option` - Закрыть опрос (с автодополнением)

## Автодополнение

Следующие команды поддерживают автодополнение:
- **Кейсы**: `/open_case`, `/buy_case_bank`, `/case_trade`, `/admin_give_case`
- **NFT**: `/sell`, `/trade_nft`, `/show_nft`, `/admin_give_nft`, `/admin_remove_nft`
- **Опросы**: `/dep`, `/admin_close_dep`

## Совместимость

- Старые команды с префиксом "!" продолжают работать
- Slash commands работают параллельно с существующими командами
- Все существующие функции сохранены

## Установка и запуск

1. Убедитесь, что у бота есть необходимые права:
   - `applications.commands` - для slash commands
   - `Send Messages` - для отправки сообщений
   - `Use Slash Commands` - для использования slash commands

2. Запустите бота как обычно:
   ```bash
   go run main.go
   ```

3. Slash commands автоматически зарегистрируются при запуске

## Технические детали

### Регистрация команд
Команды регистрируются автоматически при запуске бота в функции `RegisterSlashCommands()`.

### Обработка взаимодействий
Все slash commands обрабатываются через `HandleSlashCommand()` в `bot/slash_commands.go`.

### Автодополнение
Автодополнение реализовано через `HandleAutocomplete()` и использует данные из Redis и Google Sheets.

### Совместимость
Для совместимости с существующими обработчиками используется функция `buildCommandString()`, которая преобразует параметры slash command в строку, совместимую с старыми обработчиками.

## Преимущества Slash Commands

1. **Лучший UX**: Встроенная справка и автодополнение
2. **Валидация**: Discord автоматически проверяет типы параметров
3. **Меньше ошибок**: Пользователи не могут ввести неправильный синтаксис
4. **Современный стандарт**: Соответствует текущим рекомендациям Discord
5. **Мобильная поддержка**: Лучше работает на мобильных устройствах

## Миграция

Для полной миграции на slash commands:

1. Уведомите пользователей о новых командах
2. Обновите документацию
3. Постепенно отключите поддержку команд с "!" (опционально)

## Поддержка

При возникновении проблем:
1. Проверьте логи бота на наличие ошибок регистрации
2. Убедитесь, что у бота есть необходимые права
3. Проверьте, что команды зарегистрированы в Discord Developer Portal
